TinyMouse

Tiny Mouse 是一个 Windows 平台下的简单 HTTP 服务器，能够响应客户端静态资源请求，比如 html，图片等。

使用方法：
usage:
--ip default=0.0.0.0, listen ip address
--port default=80, listen port
--socket default=30, max listen socket number

example:
MiniWebServer.exe --port=12345

项目地址：https://github.com/jelly-lemon/MiniWebServer
